# Security Policy

## Supported Versions
We currently provide security updates for the following versions of SmartLearn:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please do not open a public issue. Instead, send an email to the project maintainers directly or use the feedback form provided in the live application. We will attempt to address the vulnerability as quickly as possible.
